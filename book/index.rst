PyGame para aprender python
=================================
Como indica el nombre, este es un curso diseñado para enseñar Python utilizando la creación de videojuegos
como metodología. El curso está estructurado para guiar a los estudiantes a través de los conceptos básicos 
de la programación con Python y PyGame, proporcionando ejemplos prácticos y ejercicios.

Objetivos del Curso
-------------------

- Aprender los fundamentos de la programación y del lenguaje Python.
- Desarrollar habilidades para crear juegos utilizando PyGame.
- Aplicar el conocimiento adquirido en proyectos prácticos.
- Entender los conceptos básicos de los motores de videojuegos.
